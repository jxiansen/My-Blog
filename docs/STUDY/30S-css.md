# 30s-css

### 使用纯css画一个圆形.

- 使用`border-radius: 50%`来改变div边框。
- 因为圆形的半径处处相等，所以 `width` 和 `height`属性必须相等，否则将会创建一个椭圆形。

```html
<div class="circle"></div>
```

```css
.circle {
  border-radius: 50%;
  width: 32px;
  height: 32px;
  background: #9C27B0;
}
```
